## 🚀 Sobre o projeto

Este repositório reúne diversos exercícios e implementações de lógica de programação, organizados de forma estruturada para demonstrar evolução, organização e capacidade de resolução de problemas.

Aqui você encontrará desde conceitos básicos até pequenas simulações de sistemas com regras de negócio.

## 💡 Destaques

Alguns projetos simulam situações reais e vão além de exercícios básicos:

- 🛒 **Shopping Calculation System**  
  Cálculo de compras com regras de pagamento, descontos e acréscimos

- 👨‍💼 **Employee Dependents System**  
  Cálculo baseado em dependentes e impacto no salário

- 📊 **Student Performance System**  
  Avaliação de desempenho de alunos com base em notas

- 💰 **Currency Converter**  
  Conversão de valores com lógica aplicada
